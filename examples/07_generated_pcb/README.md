# Generated PCB Example

This project is generated by the Go direct-file writer and includes:

- `generated_pcb.kicad_pro`
- `generated_pcb.kicad_sch`
- `generated_pcb.kicad_pcb`

Regenerate from the repository root with:

```sh
kicadai --output examples/07_generated_pcb --name generated_pcb --seed pcb-phase-9 --with-pcb --overwrite generate-project
```

Run normal validation with:

```sh
go test ./...
```

Run KiCad CLI PCB validation, when KiCad is installed, with:

```sh
KICADAI_RUN_KICAD_CLI=1 go test ./internal/kicadfiles/pcb
```
