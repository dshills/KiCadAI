# PCB Object Correctness Fixture

This example is generated by `pcb.CorrectnessFixturePCB` and exercises the
core PCB writer object families:

- SMD and through-hole footprints with embedded geometry.
- SMD and through-hole pads with declared nets.
- Front and back copper tracks.
- A route arc.
- Vias.
- A GND zone.
- Silkscreen text.
- A closed board outline.

Regenerate after intentional writer changes:

```sh
KICADAI_UPDATE_FIXTURES=1 go test ./internal/kicadfiles/pcb -run TestCheckedInCorrectnessFixtureMatchesGenerator -count=1
```

Validate the checked-in fixture:

```sh
go test ./...
```

Optional KiCad CLI round-trip checks can be enabled with the round-trip
environment gates documented in `specs/pcb-writer/PCB_OBJECT_CORRECTNESS_PLAN.md`.
